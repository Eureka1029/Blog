## 构造函数

构造函数是一种特殊类型的方法，主要就是用在这里，每当你创造一个对象的时候就会被调用

```cpp
class Entity
{
public:
    float X, Y;

    Entity()   // 构造函数
    {
        X = 0.0f;
        Y = 0.0f;
    }

    void Print()
    {
        std::cout << X << "," << Y << std::endl;
    }
};

int main()
{
    Entity e;
    std::cout << e.X << std::endl;  // 0
    e.Print();  // 0,0
    std::cin.get();
};
```

###  默认构造函数

> default construction是默认就有的，大约就像空函数体那样，什么都没做

```cpp
Entity()  // default construction 类似
{
}
```

没有初始化变量。

C++中必须手动初始化**所有的基本类型**，不然它们会被设置为之前留存在内存中的值

### 隐藏构造函数 

> 如果我只想让别人调用类的静态方法,而不允许它创建类实例,有以下两种方法

1. private隐藏构造函数
```cpp
class Log
{
private:
    Log(){}
public:
    static void Write()
    {
    }
};

int main()
{
    Log::Write();
    Log l;
}
```


2. delete
```cpp
Log() = delete;
```
删除默认构造函数
