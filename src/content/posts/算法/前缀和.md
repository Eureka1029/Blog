## §1.1 基础
模板题:
- [x] [303. 区域和检索 - 数组不可变](https://leetcode.cn/problems/range-sum-query-immutable/)
```cpp
class NumArray {
    vector<int> s;
public:
    NumArray(vector<int>& nums) {
        s.resize(nums.size() + 1);
        for(int i = 0; i < nums.size(); i++){
            s[i + 1] = s[i] + nums[i];
        }
    }
    
    int sumRange(int left, int right) {
        return s[right + 1] - s[left];
    }
};

```
