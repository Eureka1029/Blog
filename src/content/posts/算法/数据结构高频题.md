## 设计有SetAll功能的哈希表

https://www.nowcoder.com/practice/7c4559f138e74ceb9ba57d76fd169967

哈希表常见的三个操作时put、get和containsKey，而且这三个操作的时间复杂度为O(1)。现在想加一个setAll功能，就是把所有记录value都设成统一的值。请设计并实现这种有setAll功能的哈希表，并且put、get、containsKey和setAll四个操作的时间复杂度都为O(1)。 

[友情提示]: C++选手若有需要可以使用unordered_map替换map来将复杂度从O(log n)降为O(1)


> 本题使用时间戳技巧实现,基本思路是有个计时器cnt,有个setAllTime变量用来记录setAll操作发生的时间,在哈希表里多记录一项存入时间.这样取数据的时候只需要对比存入时间和setAllTime时间,即可判断返回原值还是setAllValue的值



```cpp
#include <bits/stdc++.h>
using namespace std;
map<int,pair<int, int>> mp;
int setAllTime; // 记录SetAll操作发生的时间
int setAllValue; //记录SetAll操作的值
int cnt; // 计时器(操作记录器)


void put(int k, int v){
    if(mp.find(k) != mp.end()){ //没存在哈希表
        mp[k].first = v; 
        mp[k].second = cnt++; // 记录存入操作的时间
    }else{ // 已经存入哈希表了,修改值和操作时间
        mp[k] = {v, cnt++};
    }
}


int get(int k){ //获得值
    if(mp.find(k) != mp.end()){
        if(mp[k].second > setAllTime){ // 晚于SetAll操作
            return mp[k].first; //正常返回值
        }else{ //SetAll操作直接就存入了
            return setAllValue;  // 返回覆盖的值
        }
    }else{
        return -1; //没找到
    }
}
void setAll(int v){
    setAllValue = v; //设置覆盖值
    setAllTime = cnt++; //记录当前操作的值
}
int main() {
    int n, op, a, b;
    ios::sync_with_stdio(false);
    cin.tie(0);
    
    while(cin >> n){
	    // 初始化变量
        mp.clear();
        setAllTime = -1;
        setAllValue = 0;
        cnt = 0;
        
        
        for(int i = 0; i < n; i++){
            cin >> op;
            if(op == 1){
                cin >> a >> b;
                put(a, b);
            }else if(op == 2){
                cin >> a;
                cout << get(a) << "\n";
            }else if(op == 3){
                cin >> a;
                setAll(a);
            }
        }
    }

    
}
```

## LRU缓存
https://leetcode.cn/problems/lru-cache/description/

请你设计并实现一个满足  [LRU (最近最少使用) 缓存](https://baike.baidu.com/item/LRU) 约束的数据结构。

实现 `LRUCache` 类：

- `LRUCache(int capacity)` 以 **正整数** 作为容量 `capacity` 初始化 LRU 缓存
- `int get(int key)` 如果关键字 `key` 存在于缓存中，则返回关键字的值，否则返回 `-1` 。
- `void put(int key, int value)` 如果关键字 `key` 已经存在，则变更其数据值 `value` ；如果不存在，则向缓存中插入该组 `key-value` 。如果插入操作导致关键字数量超过 `capacity` ，则应该 **逐出** 最久未使用的关键字。

函数 `get` 和 `put` 必须以 `O(1)` 的平均时间复杂度运行。

> 本题使用双向链表＋哈希表来实现

> 将对象和值封装成一个双向链表的结点,哈希表存对象和其对应结点的映射.
> 链表越往前来的越早,越往后来的越晚,记录头结点(最久没操作)和尾结点(最近刚操作)的地址.每当put或get就放到尾结点的后面


> 哈希表用于存取数据,双向链表用来构建时间链


```cpp
class LRUCache {
public:
	// 定义双向链表的节点
    class DoubleNode{
    public:
        int key, val; // 存key是为了后续已知节点地址,在哈希表中删除该表项.
        DoubleNode* last;
        DoubleNode* next;
        
        //构造函数
        DoubleNode(int k, int v) : key(k), val(v), last(nullptr), next(nullptr) {}
    };


    class DoubleList{
    private:
        DoubleNode* head; // 头节点指针,用于移除节点
        DoubleNode* tail; // 尾节点指针,用于插入新的节点.
    public:
	    // 构造函数
        DoubleList() : head(nullptr), tail(nullptr) {}
        
        // 添加新的节点
        void addNode(DoubleNode* newNode) {
	        //特判
            if(newNode == nullptr) return;
            
            if(head == nullptr){
	            //链表为空
                head = tail = newNode;
            } else {
	            //链表不为空
                tail->next = newNode;
                newNode->last = tail;
                tail = newNode;
            }
        }
        
        //移动已有节点到最后
        void moveNodeToTail(DoubleNode* node) {
	        //特判
            if(tail == node) return;
            
            
            if(head == node){
	            //头节点由于head->last为空,如果node == head,需要特殊讨论
                head = node->next;
                head->last = nullptr;
            }else{
	            //断开当前节点
                node->last->next = node->next;
                node->next->last = node->last;
            }
            //连到tail节点之后
            node->last = tail;
            node->next = nullptr;
            tail->next = node;
            tail = node;
        }
        
        //移除头节点(最久未使用节点)
        DoubleNode* removeHead(){
	        //特判
            if(head == nullptr) return nullptr;
            //需要返回的节点,需要返回节点指针的原因是,我们在创建指针是在堆上进行的,因此需要把地址穿出去,delete释放掉.
            DoubleNode* ans = head;
            
            if(head == tail){
	            //此时链表只有一个节点
                head = tail = nullptr;
            }else{
	            //断开头节点
                head->next->last = nullptr;
                head = head->next;
                ans->next = nullptr;
            }
            return ans;
        }
    };

    unordered_map<int, DoubleNode*> keyNodeMap;
    DoubleList nodeList;
    int cap;

    LRUCache(int capacity) : cap(capacity){}
    
    int get(int key) {
        if(keyNodeMap.find(key) != keyNodeMap.end()){
	        //在哈希表中,把该节点放入链表尾
            nodeList.moveNodeToTail(keyNodeMap[key]);
            return keyNodeMap[key]->val;
        }
        //没找到节点
        return -1;
    }
    
    void put(int key, int value) {
        if(keyNodeMap.find(key) != keyNodeMap.end()){
	        //在哈希表中,把该节点放入链表尾
            nodeList.moveNodeToTail(keyNodeMap[key]);
            keyNodeMap[key]->val = value; //更新值
        }else{
            if(keyNodeMap.size() == cap){
	            //已经超出容量了
                DoubleNode* rm = nodeList.removeHead(); //将要删除的节点
                keyNodeMap.erase(rm->key); //哈希表中删除
                delete rm; //堆中释放内存
            }
            //加入新的节点
            DoubleNode* newNode = new DoubleNode(key,value);
            keyNodeMap[key] = newNode;
            nodeList.addNode(newNode);
        }
    }
};

```

## O(1) 时间插入、删除和获取随机元素
https://leetcode.cn/problems/insert-delete-getrandom-o1

实现`RandomizedSet` 类：

- `RandomizedSet()` 初始化 `RandomizedSet` 对象
- `bool insert(int val)` 当元素 `val` 不存在时，向集合中插入该项，并返回 `true` ；否则，返回 `false` 。
- `bool remove(int val)` 当元素 `val` 存在时，从集合中移除该项，并返回 `true` ；否则，返回 `false` 。
- `int getRandom()` 随机返回现有集合中的一项（测试用例保证调用此方法时集合中至少存在一个元素）。每个元素应该有 **相同的概率** 被返回。

你必须实现类的所有函数，并满足每个函数的 **平均** 时间复杂度为 `O(1)` 。

> 本题使用动态数组+map实现,map记录值和在数组中的下标的映射,动态数组用于实现getRandom方法,只需要在0-size中求随机值即可.map用于记录是否出现过.

```cpp
class RandomizedSet {
private:
    unordered_map<int, int> map;  // 存储值和对应的索引
    vector<int> arr;              // 存储所有插入的值
public:
    RandomizedSet() {
        srand(time(0)); //初始化种子
    }
    
    bool insert(int val) {
        if(map.find(val) != map.end()){
	        //在集合中出现
            return false;
        }
        //添加元素,并记录下标
        arr.push_back(val);
        map[val] = arr.size() - 1;
        return true;
    }
    
    bool remove(int val) {
        if(map.find(val) == map.end()){
            return false;
        }
        
        int valIndex = map[val]; //获得将要修改的值的下标,我们要把它从数组中踢出去.
        int endValue = arr.back(); //获得数组最后一个元素的值,我们要把它插入到被移除的位置
		
		//插入空位置,目的是让数组连续
        arr[valIndex] = endValue;
        //改变数组末尾值的下标记录
        map[endValue] = valIndex;
        
        //删除条目
        map.erase(val);
        arr.pop_back();
        return true;
    }
    
    int getRandom() {
	    //返回随机下标.
        int randIndex = rand() % arr.size();
        return arr[randIndex];
    }
};
```

## O(1) 时间插入、删除和获取随机元素 - 允许重复
https://leetcode.cn/problems/insert-delete-getrandom-o1-duplicates-allowed/

`RandomizedCollection` 是一种包含数字集合(可能是重复的)的数据结构。它应该支持插入和删除特定元素，以及删除随机元素。

实现 `RandomizedCollection` 类:

- `RandomizedCollection()`初始化空的 `RandomizedCollection` 对象。
- `bool insert(int val)` 将一个 `val` 项插入到集合中，即使该项已经存在。如果该项不存在，则返回 `true` ，否则返回 `false` 。
- `bool remove(int val)` 如果存在，从集合中移除一个 `val`项。如果该项存在，则返回 `true` ，否则返回 `false` 。注意，如果 `val` 在集合中出现多次，我们只删除其中一个。
- `int getRandom()` 从当前的多个元素集合中返回一个随机元素。每个元素被返回的概率与集合中包含的相同值的数量 **线性相关** 。

您必须实现类的函数，使每个函数的 **平均** 时间复杂度为 `O(1)` 。

**注意** : 生成测试用例时，只有在 `RandomizedCollection` 中 **至少有一项** 时，才会调用 `getRandom` 。

> 本题只需要将上题的映射改为对set的映射即可,set记录该变量所在的所有下标.

```cpp
class RandomizedCollection {
private:
    vector<int> arr;
    unordered_map<int, unordered_set<int>> mp;

public:
    RandomizedCollection() {
        srand(time(0)); //初始化种子
    }
    
    bool insert(int val) {
        unordered_set<int>& st = mp[val];
        st.insert(arr.size()); 
        arr.push_back(val);
        return st.size() == 1;
    }
    
    bool remove(int val) {
        if(mp.find(val) == mp.end()){
            return false;
        }
        unordered_set<int>& valSet = mp[val];
        int endValue = arr.back();
        int valAnyIndex = *(valSet.begin());

        if (val == endValue) {
            valSet.erase(arr.size() - 1);  // 如果值是最后一个元素，直接删除
        } else {
            unordered_set<int>& endValueSet = mp[endValue];
            endValueSet.insert(valAnyIndex);  // 将最后一个元素的索引添加到 val 的位置
            arr[valAnyIndex] = endValue;  // 将最后一个元素移到要删除的元素位置
            endValueSet.erase(arr.size() - 1);  // 删除最后一个元素
            valSet.erase(valAnyIndex);  // 删除要删除的元素
        }

        arr.pop_back();  // 删除最后一个元素
        if (valSet.empty()) {
            mp.erase(val);  // 如果该值的所有索引都被删除，移除该值
        }

        return true;
    }
    
    int getRandom() {
        int randIndex = rand() % arr.size();
        return arr[randIndex];
    }
};
```
