## 相交链表
https://leetcode.cn/problems/intersection-of-two-linked-lists/

![](../../../images/截屏2026-03-12%2015.45.23.png)


>本题相对简单,最简单的做法是直接找两个容器,分别记录下两个链表的所有节点,再对其中一个链表进行遍历,看是否存在于另一个容器之中即可解决,本文不使用该方法,有兴趣的读者可自行实现


> [!note] 
> **解决思路**
> 我们可以记录每个链表分别有多少个节点，找出较长的链表，让该链表的头指针先走节点之差的步数，两个指针再一起向前走即可在相交点相遇。



**示例代码**
```cpp
ListNode *getIntersectionNode(ListNode *headA, ListNode *headB){
	if(headA == nullptr || headB == nullptr){
		return nullptr;
	}
	int diff = 0;
	ListNode* a = headA;
	ListNode* b = headB;
	while(a->next != nullptr){
		a = a->next;
		diff++;
	}
	while(b->next != nullptr){
		b = b->next;
		diff--;
	}
	if(a != b){
		return nullptr;
	}
	if(diff >= 0){
		a = headA;
		b = headB;
	}else{
		b = headA;
		a = headB;
	}
	diff = abs(diff);
	while(diff-- != 0){
		a = a->next;
	}
	while(a != b){
		a = a->next;
		b = b->next;
	}
	return a;
}
```


### 代码详解

```cpp
if(headA == nullptr || headB == nullptr){
	return nullptr;
}
```
>用一个变量来记录链表节点差的实现,分别遍历两个链表,以下一个节点不为空作为while循环判断条件是为了在下面进行特判



```cpp
int diff = 0;
ListNode* a = headA;
ListNode* b = headB;
while(a->next != nullptr){
	a = a->next;
	diff++;
}
while(b->next != nullptr){
	b = b->next;
	diff--;
}
```
>用一个变量来记录链表节点差的实现,分别遍历两个链表,以下一个节点不为空作为while循环判断条件是为了在下面进行特判



```cpp
if(a != b){
	return nullptr;
}
if(diff >= 0){
	a = headA;
	b = headB;
}else{
	b = headA;
	a = headB;
}
```
>特判两个链表不相交,相交则一定同一终点,这里使用a指针来指向较长链表的头节点,b指针指向较短链表的头节点



```cpp
diff = abs(diff);
while(diff-- != 0){
	a = a->next;
}
while(a != b){
	a = a->next;
	b = b->next;
}
return a;
```
>让a指针先走diff步,最后两个指针一起向后遍历直到相遇,返回任一指针都行

## K个一组翻转链表
https://leetcode.cn/problems/reverse-nodes-in-k-group/description/
![](../../../images/截屏2026-03-12%2016.29.51.png)

>本题简单做法是使用容器将所有节点装入,在容器中进行翻转,最后再把容器中的元素用链表串起来.


> [!NOTE]   
>  **解决思路**   
> 本题思路是,对每一组进行翻转,并让翻转后的尾节点指向下一组的头节点,这样就能找到我们要继续翻转的下一组,翻转下一组之后,再让尾节点指向刚翻转完的新头节点.
> 

## 代码详解

从s节点开始,找到这一组的最后一个节点:
> - -k 的含义是:上一个循环s所在的这个节点不是空,那说明找到了一个节点,还要找- -k个节点,如果还要找的节点个数为0那就不用找了.
> s != nullptr 的含义是,如果还有要找的节点,但是s当前循环yi j是空2了,说明已经找不到了.
```cpp
ListNode* teamEnd(ListNode* s, int k){
	while(--k != 0 && s != nullptr){
		s = s->next;
	}
	return s;
}
```



翻转开头为s,结尾为e的链表,并让翻转后的尾节点指向下一组的头节点.
> next指针保证记得下一个要翻转的节点,pre用来记录cur准备指向的节点,while的判断条件是cur不等于该组最后一个节点的next,但是我们一开始就让e = e->next,所以是 cur != e
```cpp
void reverse(ListNode* s, ListNode* e){
	e = e->next;
	ListNode* pre = nullptr;
	ListNode* cur = s;
	ListNode* next = nullptr;
	while(cur != e){
		next = cur->next;
		cur->next = pre;
		pre = cur;
		cur = next;
	}
	s->next = e;
}
```



解决问题:
>lastEndNode用来指向上一组的结尾,因为下一组翻转后,我们要把上一组的结尾链接上下一组的开头,也就是翻转后的end指针. 头节点因为要返回,比较特殊,所以不在循环里处理. 总体逻辑就是 : 分组 → 翻转 → 链接
```cpp
ListNode* reverseKGroup(ListNode* head, int k) {
	ListNode* start = head;
	ListNode* end = teamEnd(start, k);
	if(end == nullptr){
		return head;
	}
	reverse(start, end);
	head = end;
	ListNode* lastEndNode = start;
	while(lastEndNode->next != nullptr){
		start = lastEndNode->next;
		end = teamEnd(start, k);
		if(end == nullptr){
			return head;
		}
		reverse(start, end);
		lastEndNode->next = end;
		lastEndNode = start;
		
	}
	return head;
}
```

## 复制带随机指针的链表
https://leetcode.cn/problems/copy-list-with-random-pointer/description/

给你一个长度为 n 的链表，每个节点包含一个额外增加的随机指针 random ，该指针可以指向链表中的任何节点或空节点。

构造这个链表的 深拷贝。 深拷贝应该正好由 n 个 全新 节点组成，其中每个新节点的值都设为其对应的原节点的值。新节点的 next 指针和 random 指针也都应指向复制链表中的新节点，并使原链表和复制链表中的这些指针能够表示相同的链表状态。复制链表中的指针都不应指向原链表中的节点 。

例如，如果原链表中有 X 和 Y 两个节点，其中 X.random --> Y 。那么在复制链表中对应的两个节点 x 和 y ，同样有 x.random --> y 。

返回复制链表的头节点。

>本题也可以使用map来完成,将节点与copy节点一一对应,再使用迭代器对map进行遍历,copy节点的所有指针都可以由其对应的节点来给出,笔者自行写了一个可以通过测试的代码用作参考,请思考如何不使用容器来实现.

```cpp
class Solution {
public:
    Node* copyRandomList(Node* head) {
        map<Node*,Node*> mp;
        Node* start = head;
        while(start != nullptr){
            mp[start] = new Node(start->val);
            start = start->next;
        }
        for(map<Node*,Node*>::iterator it = mp.begin(); it != mp.end(); ++it){
            it->second->next = mp[it->first->next];
            it->second->random = mp[it->first->random];
        }
        return mp[head];
    }
};
```


>**解决思路**:可以在原先的链表的两个节点间插入一个copy节点,这样我们可以两个两个遍历新的链表,由前一个节点指示他的next的copy节点如何链接.

```cpp
class Solution {
public:
    Node* copyRandomList(Node* head) {
	    //特判
        if(head == nullptr){
            return nullptr;
        }
        
        
        
        //原先链表的两个节点中插入新的节点,链表节点插入不再讲解
        Node* cur = head;
        while(cur != nullptr){
            Node* copy = new Node(cur->val);
            copy->next = cur->next;
            cur->next = copy;
            cur = copy->next;
        }



		//设置random指针,next指针标识cur指针下一个要去的地方,由于random可能为空,所以要用三元运算符进行判断
        cur = head;
        Node* next = nullptr;
        while(cur != nullptr){
            next = cur->next->next;
            Node* copy = cur->next;
            copy->random = cur->random != nullptr ? cur->random->next : nullptr;
            cur = next;
        }

		//分离两个链表,next指针标识cur指针下一个要去的地方,由于是两个两个遍历,next只会指向原先就有的节点,因此next = cur->next->next; 后next为空时,说明已经遍历到了最后一个,此时copy作为复制后的最后一个节点也要指向空.
        cur = head;
        Node* ans = head->next;
        while(cur != nullptr){
            next = cur->next->next;
            Node* copy = cur->next;
            cur->next = next;
            copy->next = next != nullptr ? next->next : nullptr;
            cur = next;
        }
        return ans;

    }
};
```

## 回文链表
https://leetcode.cn/problems/palindrome-linked-list/

给你一个单链表的头节点 head ，请你判断该链表是否为回文链表。如果是，返回 true ；否则，返回 false 

> 本题使用栈十分简单,因为压栈的过程就是逆序的过程,在出栈判断是否相等即可

```cpp
class Solution {
public:
    bool isPalindrome(ListNode* head) {
        stack<ListNode*> st;
        ListNode* start = head;
        while(start != nullptr){
            st.push(start);
            start = start->next;
        }
        
        start = head;
        while(start != nullptr){
            if(st.top()->val != start->val){
                return false;
            }
            start = start->next;
            st.pop();
        }
        return true;
    }
};
```


我们要用 `O(n)` 时间复杂度和 `O(1)` 空间复杂度解决此题

**快慢指针**
>可以求出中点,快指针一次走两步,慢指针指针一次走一步,快指针走到头的时候慢指针刚好到中点.


> **解决思路**:使用快慢指针遍历找到中点,让中点以后的节点逆序,并让中点指向空(相当于终点相交的两个链表),再用两个指针双向遍历判断回文,最后还原链表.

```cpp
class Solution {
public:
    bool isPalindrome(ListNode* head) {
	    //特判
        if(head == nullptr || head->next == nullptr){
            return true;
        }
        
        //快慢指针遍历找中点
        ListNode* fast = head;
        ListNode* slow = head;
        while(fast->next != nullptr && fast->next->next != nullptr){
            slow = slow->next;
            fast = fast->next->next;
        }
		//此时slow指向中点
		
		
        //  从slow指针指向的节点开始翻转链表
        ListNode* pre = slow;
        ListNode* cur = slow->next;
        ListNode* next = nullptr;
        pre->next = nullptr;
        while(cur != nullptr){
            next = cur->next;
            cur->next = pre;
            pre = cur;
            cur = next;
        }

		
		//双向遍历一一比对,其中由于slow指针的的节点的next值为nullptr,所以两个双向指针的终点为nullptr
        bool ans = true;
        ListNode* left = head;
        ListNode* right = pre;
        while(left != nullptr && right != nullptr){
            if(left->val != right->val){
                ans = false; //发现不是回文
                break; //结束循环
            }
            left = left->next;
            right = right->next;
        }

        //把原来的链表进行复原,这时候的pre是最后一个节点.
        cur = pre->next;
        pre->next = nullptr;
        next = nullptr;
        while(cur != nullptr){
            next = cur->next;
            cur->next = pre;
            pre = cur;
            cur = next;
        }

        return ans;
    }
};
```

## 返回链表的第一个入环节点
https://leetcode.cn/problems/linked-list-cycle-ii/description/

给定一个链表的头节点  `head` ，返回链表开始入环的第一个节点。 _如果链表无环，则返回 `null`。_

如果链表中有某个节点，可以通过连续跟踪 `next` 指针再次到达，则链表中存在环。 为了表示给定链表中的环，评测系统内部使用整数 `pos` 来表示链表尾连接到链表中的位置（**索引从 0 开始**）。如果 `pos` 是 `-1`，则在该链表中没有环。**注意：`pos` 不作为参数进行传递**，仅仅是为了标识链表的实际情况。

**不允许修改** 链表

>本题如果使用容器set可以直接秒杀,进行遍历,查是否在set中出现过即可.

```cpp
class Solution {
public:
    ListNode *detectCycle(ListNode *head) {
        ListNode* start = head;
        ListNode* ans = nullptr;
        set<ListNode*> st;
        while(start != nullptr){
            if(st.count(start) == 1){
                ans = start;
                break;
            }
            st.insert(start);
            start = start->next;
        }
        return ans;
    }
};
```


**我们可以使用快慢指针解决,可以记录该问题的基本流程,这里不做证明**
1. 快指针一次走两步,慢指针一次走一步
2. 快指针如果为空——无环
3. 快慢指针相遇之后,快指针回到起始点,慢指针不动
4. 快指针和慢指针都一次走一步,相遇的点就是交点.

```cpp
class Solution {
public:
    ListNode *detectCycle(ListNode *head) {
	    //如果是空指针,或者只有一个节点的指针且指向为空的特判
        if(head == nullptr || head->next == nullptr) return nullptr;
		
		//快慢指针,快指针走的快,因此如果快指针已经是空或者发现下一个指针为空就停止循环,快指针一定要看下一个指针是不是空,因为要写fast = fast->next->next;nullptr->next会导致报错.
        ListNode* fast = head;
        ListNode* slow = head;
        while(fast != nullptr && fast->next != nullptr){
            fast = fast->next->next;
            slow = slow->next;
            if(fast == slow) break;
        }
        
		//判断是不是因为发现空而退出的循环
        if(fast == nullptr || fast->next == nullptr) return nullptr;
		
		
		//此时确认有环
        fast = head;
        while(fast != slow){
            fast = fast->next;
            slow = slow->next;
        }
		
		//返回slow和fast一样
        return slow;
        
    }
};
```

## 排序链表
https://leetcode.cn/problems/sort-list/description/

给你链表的头结点 `head` ，请将其按 **升序** 排列并返回 **排序后的链表** 。

> 本题使用归并排序的非递归写法

```cpp
class Solution {
public:
    ListNode* start;
    ListNode* end;
    ListNode* sortList(ListNode* head) {
        int n = 0;
        ListNode* cur = head;
        while (cur != nullptr) {
            n++;
            cur = cur->next;
        }
        // l1...r1 每组的左部分
        // l2...r2 每组的右部分
        // next 下一组的开头
        // lastTeamEnd 上一组的结尾
        ListNode *l1, *l2, *r1, *r2, *next, *lastTeamEnd;
        for (int step = 1; step < n; step <<= 1) {
            l1 = head;
            r1 = findEnd(l1, step);
            // 这里l2不可能为空,因为step严格小于n.
            l2 = r1->next;
            r2 = findEnd(l2, step);
            next = r2->next;
            r1->next = nullptr;
            r2->next = nullptr;
            merge(l1, r1, l2, r2);
            head = start;
            lastTeamEnd = end;
            while (next != nullptr) {
                l1 = next;
                r1 = findEnd(l1, step);
                l2 = r1->next;
                if (l2 == nullptr) {
                    lastTeamEnd->next = l1;
                    break;
                }
                r2 = findEnd(l2, step);
                next = r2->next;
                r1->next = nullptr;
                r2->next = nullptr;
                merge(l1, r1, l2, r2);
                lastTeamEnd->next = start;
                lastTeamEnd = end;
            }
        }
        return head;
    }
    ListNode* findEnd(ListNode* s, int k) {
        // 包括s在内,往下数k个节点
        // 数量不够返回最后一个非空节点
        // 说明s不能为空
        // 如果k == 0 了 就不要往下走了, 此时 s 就是最后一个了
        while (s->next != nullptr && --k != 0) {
            s = s->next;
        }
        return s;
    }

    void merge(ListNode* l1, ListNode* r1, ListNode* l2, ListNode* r2) {
        ListNode* pre;
        if (l1->val <= l2->val) {
            start = l1;
            pre = l1;
            l1 = l1->next;
        } else {
            start = l2;
            pre = l2;
            l2 = l2->next;
        }
        while (l1 != nullptr && l2 != nullptr) {
            if (l1->val <= l2->val) {
                pre->next = l1;
                pre = l1;
                l1 = l1->next;
            } else {
                pre->next = l2;
                pre = l2;
                l2 = l2->next;
            }
        }
        if (l1 == nullptr) {
            pre->next = l2;
            end = r2;
        }
        if (l2 == nullptr) {
            pre->next = l1;
            end = r1;
        }
    }
};
```
