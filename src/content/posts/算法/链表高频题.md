## 相交链表
https://leetcode.cn/problems/intersection-of-two-linked-lists/

![](../../../images/截屏2026-03-12%2015.45.23.png)


>本题相对简单,最简单的做法是直接找两个容器,分别记录下两个链表的所有节点,再对其中一个链表进行遍历,看是否存在于另一个容器之中即可解决,本文不使用该方法,有兴趣的读者可自行实现


> [!NOTE] 解决思路
> 我们可以记录每个链表分别有多少个节点,找出较长的链表,让该链表的头指针先走节点之差的步数,两个指针再一起向前走即可在相交点相遇

**示例代码**
```cpp
ListNode *getIntersectionNode(ListNode *headA, ListNode *headB){
	if(headA == nullptr || headB == nullptr){
		return nullptr;
	}
	int diff = 0;
	ListNode* a = headA;
	ListNode* b = headB;
	while(a->next != nullptr){
		a = a->next;
		diff++;
	}
	while(b->next != nullptr){
		b = b->next;
		diff--;
	}
	if(a != b){
		return nullptr;
	}
	if(diff >= 0){
		a = headA;
		b = headB;
	}else{
		b = headA;
		a = headB;
	}
	diff = abs(diff);
	while(diff-- != 0){
		a = a->next;
	}
	while(a != b){
		a = a->next;
		b = b->next;
	}
	return a;
}
```


### 代码详解

```cpp
if(headA == nullptr || headB == nullptr){
	return nullptr;
}
```
>用一个变量来记录链表节点差的实现,分别遍历两个链表,以下一个节点不为空作为while循环判断条件是为了在下面进行特判



```cpp
int diff = 0;
ListNode* a = headA;
ListNode* b = headB;
while(a->next != nullptr){
	a = a->next;
	diff++;
}
while(b->next != nullptr){
	b = b->next;
	diff--;
}
```
>用一个变量来记录链表节点差的实现,分别遍历两个链表,以下一个节点不为空作为while循环判断条件是为了在下面进行特判



```cpp
if(a != b){
	return nullptr;
}
if(diff >= 0){
	a = headA;
	b = headB;
}else{
	b = headA;
	a = headB;
}
```
>特判两个链表不相交,相交则一定同一终点,这里使用a指针来指向较长链表的头节点,b指针指向较短链表的头节点



```cpp
diff = abs(diff);
while(diff-- != 0){
	a = a->next;
}
while(a != b){
	a = a->next;
	b = b->next;
}
return a;
```
>让a指针先走diff步,最后两个指针一起向后遍历直到相遇,返回任一指针都行

## K个一组翻转链表
https://leetcode.cn/problems/reverse-nodes-in-k-group/description/
![](../../../images/截屏2026-03-12%2016.29.51.png)

>本题简单做法是使用容器将所有节点装入,在容器中进行翻转,最后再把容器中的元素用链表串起来.

> [!NOTE] 解决思路
> 本题思路是,对每一组进行翻转,并让翻转后的尾节点指向下一组的头节点,这样就能找到我们要继续翻转的下一组,翻转下一组之后,再让尾节点指向刚翻转完的新头节点.
> 

## 代码详解

从s节点开始,找到这一组的最后一个节点:
> --k 的含义是:上一个循环s所在的这个节点不是空,那说明找到了一个节点,还要找--k个节点,如果还要找的节点个数为0那就不用找了.
> s != nullptr 的含义是,如果还有要找的节点,但是s当前循环以及是空了,说明已经找不到了.
```cpp
ListNode* teamEnd(ListNode* s, int k){
	while(--k != 0 && s != nullptr){
		s = s->next;
	}
	return s;
}
```



翻转开头为s,结尾为e的链表,并让翻转后的尾节点指向下一组的头节点.
> next指针保证记得下一个要翻转的节点,pre用来记录cur准备指向的节点,while的判断条件是cur不等于该组最后一个节点的next,但是我们一开始就让e = e->next,所以是 cur != e
```cpp
void reverse(ListNode* s, ListNode* e){
	e = e->next;
	ListNode* pre = nullptr;
	ListNode* cur = s;
	ListNode* next = nullptr;
	while(cur != e){
		next = cur->next;
		cur->next = pre;
		pre = cur;
		cur = next;
	}
	s->next = e;
}
```



解决问题:
>lastEndNode用来指向上一组的结尾,因为下一组翻转后,我们要把上一组的结尾链接上下一组的开头,也就是翻转后的end指针. 头节点因为要返回,比较特殊,所以不在循环里处理. 总体逻辑就是 : 分组 → 翻转 → 链接
```cpp
ListNode* reverseKGroup(ListNode* head, int k) {
	ListNode* start = head;
	ListNode* end = teamEnd(start, k);
	if(end == nullptr){
		return head;
	}
	reverse(start, end);
	head = end;
	ListNode* lastEndNode = start;
	while(lastEndNode->next != nullptr){
		start = lastEndNode->next;
		end = teamEnd(start, k);
		if(end == nullptr){
			return head;
		}
		reverse(start, end);
		lastEndNode->next = end;
		lastEndNode = start;
		
	}
	return head;
}
```

## 复制带随机指针的链表
https://leetcode.cn/problems/copy-list-with-random-pointer/description/

给你一个长度为 n 的链表，每个节点包含一个额外增加的随机指针 random ，该指针可以指向链表中的任何节点或空节点。

构造这个链表的 深拷贝。 深拷贝应该正好由 n 个 全新 节点组成，其中每个新节点的值都设为其对应的原节点的值。新节点的 next 指针和 random 指针也都应指向复制链表中的新节点，并使原链表和复制链表中的这些指针能够表示相同的链表状态。复制链表中的指针都不应指向原链表中的节点 。

例如，如果原链表中有 X 和 Y 两个节点，其中 X.random --> Y 。那么在复制链表中对应的两个节点 x 和 y ，同样有 x.random --> y 。

返回复制链表的头节点。

>本题也可以使用map来完成,将节点与copy节点一一对应,再使用迭代器对map进行遍历,copy节点的所有指针都可以由其对应的节点来给出,笔者自行写了一个可以通过测试的代码用作参考,请思考如何不使用容器来实现.

```cpp
class Solution {
public:
    Node* copyRandomList(Node* head) {
        map<Node*,Node*> mp;
        Node* start = head;
        while(start != nullptr){
            mp[start] = new Node(start->val);
            start = start->next;
        }
        for(map<Node*,Node*>::iterator it = mp.begin(); it != mp.end(); ++it){
            it->second->next = mp[it->first->next];
            it->second->random = mp[it->first->random];
        }
        return mp[head];
    }
};
```
